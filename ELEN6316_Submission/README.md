# ELEN6316 Submission

This folder contains the final report package, supporting simulation results, and course reference material for the current-steering DAC project.

## Layout

- `docs/` - final submission PDF and project requirements.
- `report/` - main IEEE-format report source and report figures.
- `ieee_draft/` - alternate IEEE draft/sample source kept for reference.
- `results/` - simulation data and MATLAB scripts organized by measurement type.
- `latex_starter.tex` - original starter LaTeX file retained for reference.

The main report source is `report/report.tex`.
